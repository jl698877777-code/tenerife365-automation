# Tenerife365 Automation Roadmap

## Mission
Build a practical automation and audit system for Tenerife365 property operations.

## Phase 1: Core Audits
1. Hostaway performance audit.
2. Booking.com listing and cancellation audit.
3. Airbnb listing quality and ranking audit.
4. PriceLabs pricing anomaly detection.
5. Revenue and KPI dashboard.

## Priority Rules
1. Separate Javi-owned / fixed-rent revenue from management-only revenue.
2. Do not mix LAS821 platform income into Javi operating income.
3. Focus first on pricing anomalies, occupancy gaps, cancellation risk, and listing quality.
4. Avoid storing API keys, passwords, guest data, or bank details in the repository.
